# Ejercicios - Básicos

Ubíquese en la **carpeta fuente del proyecto** deseado, use:

```
cd <assignments/Proyecto>

Ej.- cd assignments/00-HelloWorld

```
Regrese a la **carpeta fuente origen de todos los proyectos**, use:

```
Ej.- cd ../../

```
